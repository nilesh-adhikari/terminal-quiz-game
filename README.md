# Python Quiz App

  This is a beginner-friendly Python quiz application built while learning programming fundamentals and working with Python projects.
  The game allows users to choose categories and difficulty levels, answer questions, and track their score with a timer system and colored terminal output.

#Features-
    Multiple quiz categories
    Difficulty levels
    Randomized questions
    Score tracking system
    Answer validation
    Question timer
    Colored terminal output using Colorama
    Questions stored using JSON

# Technologies Used-
  Python
  JSON
  Colorama Library

# How to Run
  Open the project folder in VS Code
  Open terminal
  Run the following command:
  python quiz_app.py
